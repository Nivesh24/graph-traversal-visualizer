# Pathfinding Visualizer

A web-based Pathfinding Visualizer built using Vanilla JavaScript, HTML, and CSS. The project visualizes BFS, DFS, and Dijkstra's Algorithm on a grid, helping users understand graph traversal and shortest path computation through interactive animations.

## Features

* Interactive 20 × 50 grid
* Draggable Start and Target nodes
* Draw and remove walls (obstacles)
* Weighted (mud) cells for Dijkstra's Algorithm
* Breadth-First Search (BFS) Visualization
* Depth-First Search (DFS) Visualization
* Dijkstra's Shortest Path Algorithm Visualization
* Animated node exploration
* Shortest path reconstruction and visualization
* Clear Path and Clear Board functionality
* Tool selection and UI state management


Project Structure

## Algorithms Implemented

### Breadth-First Search (BFS)

* Explores nodes level by level
* Guarantees shortest path in an unweighted grid

### Depth-First Search (DFS)

* Explores as deep as possible before backtracking
* Does not guarantee the shortest path

### Dijkstra's Algorithm

* Supports weighted cells
* Computes the minimum-cost path from source to target
